# ce_ds_list_remove
`script`
```gml
ce_ds_list_remove(list, value)
```

## Description
Removes all occurrences of the value from the list.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| list | `real` | The id of the list. |
| value | `any` | The value to remove. |
