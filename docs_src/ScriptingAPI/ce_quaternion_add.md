# ce_quaternion_add
`script`
```gml
ce_quaternion_add(q1, q2)
```

## Description
Adds the quaternions `q1`, `q2` and stores the result to `q1`.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| q1 | `array` | The first quaternion. |
| q2 | `array` | The second quaternion. |
