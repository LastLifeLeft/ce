# ce_xml_parse
`script`
```gml
ce_xml_parse(string)
```

## Description
Parses value from the string.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| string | `string` | The string to parse. |

## Returns
`real/string` Real value or a string, where XML character entities
 are replaced with their original form.
