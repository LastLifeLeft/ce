# ce_assert_is_int
`script`
```gml
ce_assert_is_int(value, msg)
```

## Description
Checks whether the given value is an integer, if not, shows the error
 message and aborts the game.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| value | `any` | The value to check. |
| msg | `string` | The error message. |
