# ce_timer_get_time_left
`script`
```gml
ce_timer_get_time_left(timer)
```

## Description
Retrieves a time left till a timer ends.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| timer | `real` | The id of the timer instance. |

## Returns
`real` Time left till the timer ends in milliseconds.
