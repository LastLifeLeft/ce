# ce_axis_to_string
`script`
```gml
ce_axis_to_string(axis, direction)
```

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| axis | `real` | The axis. |
| direction | `CE_EInputDirection` | The axis direction. |

## Returns
`string` A human readable name of the gamepad axis.
