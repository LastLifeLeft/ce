# ce_assert
`script`
```gml
ce_assert(exp, msg)
```

## Description
Shows the error message if the expression is not a `real` (`bool`) or
 equals to `0` (`false`) and aborts the game.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| exp | `any` | The expression to assert. |
| msg | `string` | The error message. |
