# ce_ds_list_add_list_unique
`script`
```gml
ce_ds_list_add_list_unique(l1, l2)
```

## Description
If the list l2 is not in the list l1, it is added to it.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| l1 | `real` | The list to add into. |
| l2 | `real` | The list to be added. |
