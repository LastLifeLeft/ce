# ce_matrix_add_componentwise
`script`
```gml
ce_matrix_add_componentwise(m1, m2)
```

## Description
Adds matrices `m1`, `m2` componentwise and stores the result to `m1`.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| m1 | `array` | The first matrix. |
| m2 | `array` | The second matrix. |
