# ce_vec3_cross
`script`
```gml
ce_vec3_cross(v1, v2)
```

## Description
Gets the cross product of the vectors `v1`, `v2` and stores it to
 `v1`.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| v1 | `array` | The first vector. |
| v2 | `array` | The second vector. |
