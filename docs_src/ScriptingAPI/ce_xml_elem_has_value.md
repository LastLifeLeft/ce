# ce_xml_elem_has_value
`script`
```gml
ce_xml_elem_has_value(element)
```

## Description
Finds out if the given element has a value.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| element | `real` | The id of the element. |

## Returns
`boolean` True if the element has a value.
