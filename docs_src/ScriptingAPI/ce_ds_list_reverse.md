# ce_ds_list_reverse
`script`
```gml
ce_ds_list_reverse(list)
```

## Description
Creates a new list with values from the given list, but in a reverse
 order.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| list | `real` | The id of the list. |

## Returns
`real` The id of the created list.
