# ce_assert_equal
`script`
```gml
ce_assert_equal(exp, val, msg)
```

## Description
Shows the error message if the expression is not equal to `val`.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| exp | `any` | The expression to assert. |
| val | `any` | The expected value. |
| msg | `string` | The error message. |
