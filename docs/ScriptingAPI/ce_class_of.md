# ce_class_of
`script`
```gml
ce_class_of(id)
```

## Description
Retrieves a class of an object.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| id | `real` | The id of the object. |

## Returns
`real` The index of the class definition script.
