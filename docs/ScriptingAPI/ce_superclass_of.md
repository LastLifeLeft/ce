# ce_superclass_of
`script`
```gml
ce_superclass_of(id)
```

## Description
Retrieves a superclass of an object.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| id | `real` | The id of the object. |

## Returns
`real` The index of the superclass definition script or `noone` if
 the object does not have a superclass.
