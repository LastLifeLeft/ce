# ce_ds_list_add_map_unique
`script`
```gml
ce_ds_list_add_map_unique(list, map)
```

## Description
If the map is not in the list, it is added to it.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| list | `real` | The list to add into. |
| map | `real` | The map to be added. |
