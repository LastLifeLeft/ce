# ce_quaternion_lengthsqr
`script`
```gml
ce_quaternion_lengthsqr(q)
```

## Description
Gets the squared length of the quaternion.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| q | `array` | An array representing the quaternion. |

## Returns
`real` The squared length of the quaternion.
