# ce_color_alpha_to_argb
`script`
```gml
ce_color_alpha_to_argb(color, alpha)
```

## Description
Converts the color and aplha into a ARGB color.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| color | `real` | The color. |
| alpha | `real` | The alpha. |

## Returns
`real` The ARGB color.
