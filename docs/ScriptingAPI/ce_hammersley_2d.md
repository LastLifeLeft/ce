# ce_hammersley_2d
`script`
```gml
ce_hammersley_2d(i, n)
```

## Description
Gets i-th point from sequence of uniformly distributed points on a
 unit square.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| i | `real` | The point index in sequence. |
| n | `real` | The total size of the sequence. |
