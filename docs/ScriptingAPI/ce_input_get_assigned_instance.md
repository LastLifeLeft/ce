# ce_input_get_assigned_instance
`script`
```gml
ce_input_get_assigned_instance(gamepad)
```

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| gamepad | `real` | The id of the gamepad. |

## Returns
`real` The id of the instance assigned to the gamepad or `noone`.
