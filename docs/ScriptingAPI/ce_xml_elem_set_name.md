# ce_xml_elem_set_name
`script`
```gml
ce_xml_elem_set_name(element, name)
```

## Description
Sets name of the element to the given value.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| element | `real` | The id of the element. |
| name | `string` | The new name of the element. |
