# ce_has_own_prop
`script`
```gml
ce_has_own_prop(id, prop)
```

## Description
Checks whether an object has own property.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| id | `real` | The id of the object. |
| prop | `string/real` | The property identifier. |

## Returns
`bool` True if the object has the property.
