# ce_quaternion_dot
`script`
```gml
ce_quaternion_dot(q1, q2)
```

## Description
Gets the dot product of the two quaternions.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| q1 | `array` | The first quaternion. |
| q2 | `array` | The second quaternion. |

## Returns
`real` The dot product of the two quaternions.
