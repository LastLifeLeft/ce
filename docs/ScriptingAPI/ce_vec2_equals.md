# ce_vec2_equals
`script`
```gml
ce_vec2_equals(v1, v2)
```

## Description
Gets whether vectors `v1` and `v2` are equal.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| v1 | `array` | The first vector. |
| v2 | `array` | The second vector. |

## Returns
`bool` True if the vectors are equal.
