# ce_matrix_to_euler
`script`
```gml
ce_matrix_to_euler(m)
```

## Description
Gets euler angles from the YXZ rotation matrix.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| m | `array` | The YXZ rotation matrix. |

## Returns
`array` An array containing the euler angles `[rotX, rotY, rotZ]`.
