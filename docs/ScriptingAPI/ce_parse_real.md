# ce_parse_real
`script`
```gml
ce_parse_real(string)
```

## Description
Parses a real number from a string.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| string | `string` | The string to parse the number from. |

## Returns
`real/string` The parsed number or NaN if the string does not
 represent a number.
