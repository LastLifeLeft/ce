# ce_array_sort
`script`
```gml
ce_array_sort(array, ascending)
```

## Description
Sorts the array.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| array | `array` | The array to sort. |
| ascending | `bool` | True to sort the values in ascending order, false for descending. |
