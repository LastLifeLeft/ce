# ce_input_set_main_gamepad
`script`
```gml
ce_input_set_main_gamepad(id)
```

## Description
Sets the gamepad with given id as the main one.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| id | `real` | The id of the gamepad. Use -1 for no gamepad. |
